Correspondente Vianopolino

O Correspondente Vianopolino é um site de notícias da minha cidade natal, Vianópolis. Neste projeto, meu principal desafio foi realizar o redesign completo do site, aplicando princípios fundamentais de acessibilidade, usabilidade e segurança, a fim de melhorar a experiência dos leitores e a confiabilidade da plataforma.

Para o desenvolvimento, utilizei o WordPress como plataforma principal, combinando o tema Astra com o plugin Spectra e Gutenberg, garantindo um design moderno, responsivo e altamente personalizável. Para reforçar a segurança e proteger os usuários, implementei o plugin Wordfence, que foi fundamental na identificação e remoção de malwares que anteriormente comprometiam a navegação.

A retirada dos malwares resultou em uma experiência de navegação significativamente mais fluida e segura, o que foi destacado pelos usuários em feedbacks positivos. Muitos elogiaram a melhoria no tempo de carregamento do site, a navegação intuitiva e o novo design.

Meu foco foi criar uma plataforma funcional e acessível para todos, com medidas adicionais de performance otimizada, segurança robusta e um layout que atende tanto aos requisitos técnicos quanto às expectativas dos usuários.

Trajetória JavaScript

O projeto Trajetória JavaScript foi criado com o objetivo de auxiliar pessoas que, assim como eu, estão iniciando na área de desenvolvimento web. JavaScript é uma das linguagens mais utilizadas no setor, e por isso, decidi desenvolver este site para compartilhar conhecimento e ajudar outros a trilharem esse caminho com mais facilidade.Durante o desenvolvimento dos exercícios, aprofundei meus conhecimentos em HTML, CSS e JavaScript. Além disso, refinei minhas habilidades no uso de frameworks. Para este projeto, optei pelo Nuxt.js juntamente com TypeScript, que proporciona uma estrutura robusta e prática, facilitando a criação e organização do projeto.

Os 10 primeiros exercícios de JavaScript foram um marco importante na minha jornada de aprendizado. Encarei alguns desafios com determinação e, ao superá-los, senti uma grande satisfação. A cada exercício concluído, minha confiança e compreensão da linguagem cresciam, tornando a experiência ainda mais enriquecedora.
